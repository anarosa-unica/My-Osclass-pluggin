Lisez ce qui suis pour comprendre comment installer la langue française pour OSClass 2.1.1

  1. Pour installer la langue française connectez-vous au panneau de configuration de votre installation de OSClass
  2. Allez sou l'onglet Languages (Langue) ensuite cliquez sur add a new language (ajouter une langue) par la suite envoyer le fichier OSClass_2_1_1_FR.zip
  3. Allez maintenant sur Tools (Outils) et par la suite cliquez sur import data (Importé des données) et envoyez le fichier OSClass_2_1_1_alerte.sql (Ceci traduire toutes les alerte en français)

Panneau de configuration de OSClass n'est pas en français !!!!

Sous OSClass 2.1.1 même après avoir choisi le français comme langue dans le panneau de configuration la langue restera l'anglais.
Pour y remédier exécuter cette commande sql dans phpmyadmin 

UPDATE  `osclass`.`oc_t_preference` SET  `s_value` =  'fr_FR' WHERE  `oc_t_preference`.`s_section` =  'osclass' AND  `oc_t_preference`.`s_name` =  'admin_language' LIMIT 1 ;

Est-ce que cette traduction fonctionne avec OSClass 2.2 et plus ?

Elle devrait, mais certain texte pourrait de pas être traduit vous avez donc 2 choix les traduire ou attendre que je termine la traduction inutile de m'envoyez un courriel !

Comment modifier la traduction ?

Vous devez modifier les fichier .po dans le répertoire de la langue absolument avec l'editeur poedit vous pouvez le télécharger gratuitement à cette adresse : http://www.poedit.net Utilisez google pour savoir comment l'utiliser !
